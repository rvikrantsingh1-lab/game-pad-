What It Does

GamePad Controller runs a Node.js server on your PC that accepts input from your phone over any network connection. The phone renders a virtual gamepad — joysticks, buttons, and touch zones — and streams real-time input to the server, which injects keyboard and mouse events directly into the OS.

→
Install Node.js
Auto-installs via winget if missing
→
Run start.bat
Server launches, QR codes appear
→
Scan & Play

V4.0 · GOD MODE

Built-In Resilience

God Mode means the server takes care of itself. Three features eliminate common failure points entirely.

Auto-Restart Loop
The start.bat script runs in a God Mode loop — if the server crashes, it restarts automatically. Zero manual intervention required.
Network Watcher
Auto-detects USB plug/unplug events and adapts the server's network binding in real time. Switch between WiFi, Ethernet, and USB tethering seamlessly.
winget Fallback
If Node.js is not installed, the script automatically installs it via winget. First-run setup is completely hands-off.
